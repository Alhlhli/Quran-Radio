## 🕌 نبذة عن المشروع

**راديو القرآن الكريم** هو تطبيق ويب خفيف الوزن مبني بـ HTML خالص، يتيح للمستخدم الاستماع إلى محطات بث القرآن الكريم مباشرة من المتصفح دون الحاجة لتثبيت أي برنامج.

المشروع مكتوب بالكامل بـ **HTML + CSS + JavaScript** في ملفات مستقلة لا تحتاج إلى سيرفر.
